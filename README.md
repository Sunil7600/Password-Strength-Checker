
# Password Strength Checker

A Python-based tool that analyzes password strength and provides suggestions for creating stronger passwords.

## Features

- Length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Common weak pattern detection

## Technologies

- Python
- Regular Expressions (Regex)

## How to Run

1. Clone the repository
2. Run:

python password_checker.py

3. Enter a password
4. View the strength analysis
